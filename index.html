博客搭建

hexo 博客搭建

  1. 创建博客目录

  2. 安装 hexo npm install hexo -g

  3. 进入博客目录

  4. 执行 hexo init 命令, 初始化目录结构

  5. 执行成功后, 启动服务 服务命令: hexo server

其他命令

  • hexo new 博客名 —-> 新建博客

  • hexo -g 编译成浏览器能够识别的文件 (html,js,css)

  • hexo d —->(deploy) 发送到git

  • npm i hexo-deployer-git (如果git未找到, 执行此命令)

hexo 目录结构

  • themes 主题目录

  • source 文章目录

  • package.json 包描述文件

  • _config.yml 博客配置文件
